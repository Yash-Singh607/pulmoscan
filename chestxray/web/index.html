Questions

FAQ

Is PulmoScan a medical device?

No. It is a research and education demo only and must never be used for clinical diagnosis or treatment decisions. Always consult a qualified clinician.

What data was the model trained on?

The public Kaggle “Chest X-Ray Images (Pneumonia)” dataset by Paul Mooney — pediatric frontal-view radiographs labeled NORMAL or PNEUMONIA.

How accurate is it?

A fully trained ResNet-50 typically reaches ~93–95% test accuracy and ~0.97+ ROC-AUC. Your exact numbers appear in the Performance section after training.

What is Grad-CAM?

Gradient-weighted Class Activation Mapping — a technique that produces a heatmap of the image regions most responsible for the model's prediction, adding interpretability.

Can I use my own model?

Yes. Point the server at any compatible checkpoint via the CXR_CHECKPOINT_PATH environment variable.
